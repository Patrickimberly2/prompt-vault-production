# Creating Customer Testimonials

### FILL-IN-THE-BLANK **PROMPTS:**

```jsx
Write 5 testimonials that sound real for **[product]** for **[audience].**
```

```jsx
Write 5 testimonials for **[product]** in a conversational way.
```

```jsx
Write 5 testimonials for **[product]** written by **[ideal client].**
```

```jsx
Write 10 testimonials for **[product]** that address these objections:
**[Objection 1]
[Objection 2]
[Objection 3].**
```

```jsx
Write 10 testimonials for **[product]** that address these objections:
**[Objection 1]
[Objection 2]
[Objection 3]**
Make sure they sound real and use the words **[ideal client]** would use.
```
