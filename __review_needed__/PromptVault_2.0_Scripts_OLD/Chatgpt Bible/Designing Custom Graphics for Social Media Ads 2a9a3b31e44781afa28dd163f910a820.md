# Designing Custom Graphics for Social Media Ads

### FILL-IN-THE-BLANK PROMPTS**:**

```jsx
Prompt 1: What kind of images would best represent **[topic]**?

Prompt 2: List the adjectives of **[the image or scene you choose].**

Prompt 3: Describe in detail **[the image or scene you choose].**

Input all that information into an AI Art Generator like Dall-E or Midjourney.
```
