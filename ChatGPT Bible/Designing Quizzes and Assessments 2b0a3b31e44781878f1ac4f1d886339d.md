# Designing Quizzes and Assessments

### FILL-IN-THE-BLANK **PROMPTS:**

```jsx
My audience is interested in **[specific topic]**. Create a quiz that helps them 
identify their **[knowledge level/personality type/preferences]** while promoting 
**[product/service]**.
```

```jsx
I want to design an assessment for **[specific purpose, e.g., evaluating fitness
levels]**. Write questions and results that drive engagement and encourage users 
to share.
```

```jsx
Our brand is built around **[specific value, e.g., sustainability, innovation, 
community]**. Suggest an engaging quiz that educates users about this value 
while highlighting our offerings **[e.g., eco-friendly products, tech solutions]**. 
Include ideas for quiz themes, sample questions, and how to tie the results 
back to our brand's unique benefits.
```
