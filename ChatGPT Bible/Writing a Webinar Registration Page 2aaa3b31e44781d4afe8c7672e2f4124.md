# Writing a Webinar Registration Page

### FILL-IN-THE-BLANK **PROMPTS:**

```jsx
I want to write the copy for the webinar registration page of my webinar titled [**title of the webinar**]. In this webinar I will teach [**topic of the webinar**]. Can you write me the copy for the webinar registration page? Use a [**adjective**] tone of voice.
```

```jsx
Here is the opt-in page copy I wrote for my webinar: [**insert copy here**]. Write the webinar registration page copy that is aligned with the opt-in page text.
```

```jsx
On [**time and date of the webinar**] I will host a live webinar called "[**title of the webinar**]" about [**topic of the webinar**]. Now write the registration page copy: please put bullet points with the benefits of the presentation and be clear about the date and time the webinar will take place.
```
