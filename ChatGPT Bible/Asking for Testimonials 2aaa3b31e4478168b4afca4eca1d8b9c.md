# Asking for Testimonials

### FILL-IN-THE-BLANK **PROMPTS:**

```jsx
I am a coach and course creator for [**target audience**]. My goal is to have a testimonial for each client I serve. To do this I need a good way to ask for testimonials. Can you help me find the best way to do it?
```

```jsx
As a [**niche**] coach I have some testimonials, but I want to get more of them. How can I ask my past and current clients for a honest testimonial?
```

```jsx
This is the message I send to my clients to ask for a testimonial: "[**insert message**]".
Can you help me optimize this message so that each one of them gives a testimonial after my service?
```
