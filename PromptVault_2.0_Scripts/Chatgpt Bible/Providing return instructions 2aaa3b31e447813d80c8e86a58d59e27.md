# Providing return instructions

### FILL-IN-THE-BLANK **PROMPTS:**

```jsx
I request detailed instructions for initiating a return process for the **[product/order]** that was purchased by **[customer name]** on **[date]**. The **[customer is experiencing/did not like] [issue with the product]**, and I would appreciate your guidance in this matter.
```

```jsx
I need your assistance in creating a thorough return policy for **[order number]**. The customer intends to return **[product(s)]** due to **[damages/no longer needing them]**. Could you provide detailed instructions for returns, including available **[refund/exchange]** options?
```

```jsx
I am currently aiding a customer who wishes to initiate a return for their **[product/order]** due to **[damaged product/size/color issue]**. I request your assistance in generating comprehensive return instructions, including any necessary **[documents/proof of purchase]** to facilitate the return process smoothly.
```
